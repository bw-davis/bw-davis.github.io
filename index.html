Projects

High Order Calculator Grammar in C

I did a summer research project for the CIS Department Head. The goal was to take a chapter from Kernighan and Pike's The Unix Programming Environment that covers writing your own grammar and reformat the code. A lot has changed since the book was written in 1984 and there hasn't been any updates to the book. Hence, the code in it's current form wouldn't compile or run properly. The language created is called high order calculator but it in fact is fully functional, statically allocated, call by value language. Here is a link to the github repository. The YACC grammar is in hoc.y and all of the other files I created to hold the supporting functions required for the language to operate.
